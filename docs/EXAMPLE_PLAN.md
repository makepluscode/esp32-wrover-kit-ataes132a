## 예제별 상세 계획

### 예제 1: Info 명령어 ✅ (완료)
**목표**: AES132 디바이스의 기본 정보를 읽어오는 방법을 학습합니다.
**사용 명령어**: `AES132_INFO`
**학습 내용**:
- 디바이스 정보 읽기
- 응답 패킷 구조 이해
- 기본 통신 확인

**예상 코드 라인**: ~70줄

---

### 예제 2: 메모리 읽기 (BlockRead)
**목표**: AES132의 메모리 영역에서 데이터 읽기
**사용 명령어**: `AES132_BLOCK_READ`
**학습 내용**:
- 메모리 주소 지정 방법
- Word address 개념
- 데이터 읽기 패턴

**예상 코드 라인**: ~80줄

---

### 예제 3: 메모리 쓰기
**목표**: AES132의 메모리 영역에 데이터 쓰기
**사용 명령어**: `aes132m_write_memory`
**학습 내용**:
- 쓰기 권한 확인
- 메모리 영역별 쓰기 규칙
- 쓰기 후 검증

**예상 코드 라인**: ~90줄

---

### 예제 4: 랜덤 숫자 생성
**목표**: AES132의 하드웨어 RNG 사용
**사용 명령어**: `AES132_RANDOM`
**학습 내용**:
- 하드웨어 RNG의 장점
- 랜덤 바이트 생성
- 랜덤 데이터 활용

**예상 코드 라인**: ~70줄

---

### 예제 5: 카운터
**목표**: 카운터 값 읽기 및 증가
**사용 명령어**: `AES132_COUNTER`
**학습 내용**:
- 카운터 슬롯 개념
- 카운터 값 읽기
- 카운터 증가 방법
- 카운터 제한값
- 카운터 활용 사례

**예상 코드 라인**: ~100줄

---

### 예제 6: 암호화 (Encrypt)
**목표**: AES-128 암호화 수행
**사용 명령어**: `AES132_ENCRYPT`
**학습 내용**:
- AES 암호화 기본
- 키 슬롯 사용
- 암호화 모드 이해
- 평문 → 암호문 변환

**예상 코드 라인**: ~120줄

---

### 예제 7: 복호화 (Decrypt)
**목표**: AES-128 복호화 수행

**학습 내용**:
- AES 복호화 기본
- 암호문 → 평문 변환
- 복호화 검증

**예상 코드 라인**: ~120줄

**학습 내용**:

## 구현 전략

### 1. 공통 유틸리티 함수 생성
- `print_hex()`: 바이트 배열을 16진수로 출력
- `print_response()`: 응답 패킷 분석 및 출력
- `check_error()`: 에러 코드 해석 및 출력
- `init_aes132()`: AES132 초기화 공통 함수

### 2. 예제 구조
각 예제는 다음 구조를 따릅니다:
```cpp
1. 헤더 및 설정
2. 초기화 함수
3. 메인 기능 함수
4. setup() - 초기화 및 실행
5. loop() - 반복 실행 (필요시)
```

### 3. 문서화
- 각 예제마다 상세한 주석
- README 파일에 사용법 설명
- 데이터시트 참조 섹션

### 4. 점진적 난이도 증가
- 각 예제는 이전 예제의 개념을 활용
- 복잡도가 점진적으로 증가
- 실전 활용 가능한 예제 제공
